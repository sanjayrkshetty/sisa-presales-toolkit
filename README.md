# SISA Pre-Sales Toolkit

Templates for pre-sales security consulting — discovery, scoping, proposals, and threat modelling.
Built from real work at [SISA Information Security](https://www.sisainfosecurity.com).

## Contents

| File | What it is |
|------|-----------|
| `discovery/brief-template.md` | Discovery brief: client context, pain points, service fit |
| `scoping/effort-estimation.md` | Task-level scoping with assumptions and exclusions |
| `proposals/proposal-template.md` | Full proposal: exec summary → scope → timeline → pricing |
| `threat-models/STRIDE-template.md` | STRIDE threat model with AI/LLM extensions |

## Service lines covered

- **Compliance & Certification:** ISO/IEC 27001, PCI DSS v4.0/SSF, SOC 1/SOC 2, HIPAA
- **Security Testing:** Pen testing (network, web app, mobile, API, cloud, red team), VA&M
- **DFIR:** Incident response retainer, digital forensics, tabletop exercises, threat hunting

---

*Use these as starting points. Every engagement needs customisation.*
